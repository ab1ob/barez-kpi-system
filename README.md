# 📊 نظام مؤشرات الأداء — بارز

نظام مؤشرات أداء (KPI) متكامل مبني بالكامل في Excel لقياس أداء المنشأة عبر الأقسام.

## المنهجية

مبني على مبدأ: **المؤشر يُبنى من القيمة والأثر، لا من الرقم المجرد** — التوثيق والالتزام الإداري شرط جودة، والمخرجات هي ما يُقاس.

## المحتوى

- **لوحة قيادة** موحّدة بنظرة شاملة على أداء المنشأة
- **دليل مؤشرات** موثّق لكل مؤشر: التعريف، طريقة الحساب، مستويات التقييم
- أوراق مستقلة لكل قسم:
  - التسويق والمبيعات
  - استقطاب المواهب
  - التعلم والتطوير
  - نجاح العملاء
  - تطوير البرمجيات

## الاستخدام

افتح `نظام_مؤشرات_الأداء_بارز.xlsx` — يبدأ من ورقة «لوحة القيادة»، وورقة «المنهجية» تشرح طريقة البناء والتقييم.
